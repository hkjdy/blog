Linux常用操作整理(添加中...)
======

for用法
------
- 打印1-10
	- `for i in {1..10}; do echo $i;done`
	- `for ((i=1;i<=10;i++));do echo $i;done`

- 打印当前路径下所有文件以及文件夹
	- ```for i in `ls`; do echo $i; done ```

---

查看所有java进程的路径
------

```
for i in `ps aux | grep java | grep -v "grep java" | awk '{print $2}'`;do printf "%-10d"  $i;cat /proc/$i/environ |tr '\0' '\n' | grep PWD;done
```
